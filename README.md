# Replication Code for "Unpacking Complexity in Disaster Risk Reduction"
This repository contains replication code for the article:
__Lindersson et al. (2025).__ _Unpacking Complexity in Disaster Risk Reduction._ Preprint forthcoming.

## Repository Overview
This repository includes R notebooks for generating the statistical analysis and figures presented in the article. To enhance readability, corresponding markdown files (`.md`) are provided as well.

## Data Requirements
To replicate the results from scripts numbered as `02` and `03`, ensure you have the `source_data`-files located in the main directory. 
To run script `01`, the following raw data files are needed: 
+ __DRR Case Data__: A raw table of collected disaster risk reduction cases with ratings. This can be shared upon request.
+ __Geospatial Data__: Shapefiles from [GADM](https://gadm.org/). This study used GADM v3.6, downloaded on _2024-10-03_.
